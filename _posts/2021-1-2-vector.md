### Vector

- ArrayList와 동일한 구조를 갖는다.
- Vector는 자동 동기화를 보장하므로 멀티 스레드 환경에서 안정적으로 사용이 가능하다.
- 기본값은 0

```java
List<String> list = new Vector<String>();
Vector<String> vt=new Vector<String>();

vt.add(10); //메소드는 arrayList와 동일하다
```
