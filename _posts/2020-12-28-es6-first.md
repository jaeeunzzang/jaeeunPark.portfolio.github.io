`ES6`

```javascript
var arr = [1, 2, 3];
var [x, y] = arr;
/*할당문의 좌변에서 사용하여 원래 변수에서 어떤 값을 분해해 할당할지 정의.*/
console.log(x); //1
console.log(y); //2

var [x, , y] = arr;
console.log(x); //1
console.log(y); //3
```
