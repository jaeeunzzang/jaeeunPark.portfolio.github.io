### 배열 - Array

크기가 정해져있다.
단순하고 메모리를 적게 사용.
인덱스를 이용한 빠른 데이터 조회!

```java
int [] arr=new int[5]; //크기가 5인 배열 선언
for(int i=0;i<arr.length;i++){
arr[i]=i; //arr의 i번째 인덱스에 i값 할당.
}
```
